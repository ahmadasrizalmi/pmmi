Konsep Dasar Export Video

video_settings

Codec

arrow_right H.264: Standar industri, kompatibilitas tinggi, MP4
arrow_right H.265 (HEVC): Efisiensi bitrate lebih baik, ukuran file lebih kecil
arrow_right ProRes/DNxHD: Kualitas tinggi untuk editing, ukuran file besar
folder

Container (Format File)

arrow_right MP4 (.mp4): Paling umum, kompatibel dengan semua platform
arrow_right MOV (.mov): Format Apple, kualitas tinggi, ukuran besar
arrow_right AVI (.avi): Format lama, ukuran file sangat besar, tidak direkomendasikan
speed

Bitrate (Kecepatan Data)

arrow_right CBR (Constant): Bitrate konstan, ukuran file dapat diprediksi
arrow_right VBR (Variable): Kualitas optimal, ukuran file lebih kecil, disarankan
arrow_right 2-pass encoding: Analisis dua tahap untuk kualitas VBR maksimal
hd

Resolusi

arrow_right 720p (HD): 1280×720, ukuran file kecil, kualitas standar
arrow_right 1080p (Full HD): 1920×1080, kualitas tinggi, kompatibilitas luas
arrow_right 4K (UHD): 3840×2160, kualitas ultra tinggi, ukuran file besar

Resolusi & Aspect Ratio

1920×1080 (Full HD) atau 3840×2160 (4K)

Jangan ubah frame rate asli video Anda

4K: 35-68 Mbps

Gunakan VBR 2-pass untuk kualitas maksimal
